---

## Добавьте встречу

```text
/meeting
Мы решили использовать Gemini для первого демо.
Бактияр проверит документы завтра.
Риск: live-call сценарий ещё нужно проверить вручную.
```

Rhapsody сохраняет meeting, создаёт summary, tasks, decisions, risks и memory chunks.

## Задайте вопрос

```text
/ask Что мы решили по AI provider?
```

Ожидаемый результат: ответ должен ссылаться на источник из сохранённой встречи. Если проект не выбран, bot попросит выполнить `/projects` или `/new_project Название`.

## Проверьте производные данные

```text
/tasks
/decisions
/audit
```

Эти команды читают данные только из активного workspace.
