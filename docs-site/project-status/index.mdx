## Реализовано в коде

- FastAPI backend с modular routes.
- SQLAlchemy models и Alembic migrations.
- Telegram bot command handlers.
- Workspace isolation foundation.
- Meeting/document/memory/task/decision/audit domain services.
- Provider key encryption.
- Local Whisper/OpenAI STT configuration.

- Docker Compose stack.
