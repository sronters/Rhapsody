---

В Telegram-командах проект — это user-facing название для `workspace`.

## Создание

```text
/new_project Alpha
```

Ожидаемый результат: Rhapsody создаёт organization/workspace/user при необходимости, добавляет пользователя в workspace и выбирает проект активным для текущего private chat.

## Просмотр проектов

```text
/projects
```

Команда показывает проекты, доступные текущему пользователю или чату.

## Выбор проекта

```text
/use_project Alpha
```

Можно выбирать по номеру из списка, id или названию. Если проект не найден, bot возвращает сообщение с просьбой использовать `/projects`.

<Info>
Одинаковые названия проектов не объединяют память. Важен `workspace_id`, membership и привязка чата.
</Info>
