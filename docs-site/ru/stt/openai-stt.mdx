---
title: "OpenAI STT"
description: "Режим распознавания через OpenAI API."
---

```env
STT_MODE=openai
OPENAI_API_KEY=sk-DEMO_KEY_DO_NOT_USE
```

Используйте этот режим, если качество или скорость local Whisper недостаточны.

<Warning>
Демонстрационный ключ выше нерабочий. Настоящие ключи не должны попадать в docs, git или логи.
</Warning>
