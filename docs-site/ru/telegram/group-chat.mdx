---
title: "Групповой чат"
description: "Как группа привязывается к workspace."
---

Группа привязывается к одному активному workspace. После привязки сообщения, встречи и документы группы сохраняются в память этого workspace.

```text
/setup
/projects
/use_project Alpha
```

## Правила безопасности

- Telegram-группа не даёт автоматического доступа к памяти.
- Пользователь должен быть member workspace.
- Случайный участник не должен перепривязать уже связанную группу к своему проекту.
- Group hijack защита покрыта автоматическими regression tests, но реальная проверка со вторым Telegram user ещё pending.

<Warning>
Перед записью звонка участники группы должны понимать, что Rhapsody Recorder подключается к звонку и сохраняет audio chunks для STT.
</Warning>
