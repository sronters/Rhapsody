---
title: "Личный чат"
description: "Project selection для одного Telegram user."
---

Личный чат удобен для первого запуска. Пользователь создаёт проекты, переключается между ними и загружает встречи или документы.

```text
/setup
/new_project Alpha
/meeting
/ask Что решили?
```

## Поведение

- Активный проект хранится для конкретного пользователя.
- `/ask`, `/tasks`, `/decisions`, `/document`, `/audit` используют выбранный workspace.
- Если проект не выбран, bot возвращает сообщение `Сначала выберите проект`.

## Частые ошибки

- Нет проекта: выполните `/projects` или `/new_project Alpha`.
- Не найден проект: проверьте список через `/projects`.
- AI provider не настроен: проверьте `AI_MODE` и ключ provider.
